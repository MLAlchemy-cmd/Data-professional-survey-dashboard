Overview

This project focuses on analyzing survey data of data professionals using Power BI. The objective is to transform raw survey data into clear, interactive visual insights that help understand industry trends related to roles, salaries, tools, and career satisfaction.

Compared salary trends across different job roles

Analyzed popular tools and technologies used in the industry

Visualized overall career satisfaction among data professionals
